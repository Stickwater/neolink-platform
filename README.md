# 无人车产业与教育融合平台

## 项目简介

这是一个专注于无人车产业与教育融合的官方网站，包含数据看板、人员培训认证、需求互动、生态板块四大核心功能模块。

## 核心功能

### 1. 数据看板
- 省级地图可视化展示各省份合作院校分布
- 三级合作等级分类（待合作、已有合作、深度合作）
- 点击查看详细信息：合作院校、学生数量、岗位分布
- 实时数据统计卡片

### 2. 人员培训和认证
- 在校学生培训模块展示
- 社会人员就近申请岗位通道
- 课程资源（需登录访问）
- 认证体系（需登录访问）
- 实训基地信息

### 3. 需求侧互动
- 代理商专属留言通道
- 需求提交、意见反馈、咨询事宜
- 常见问题解答
- 留言记录展示
- 联系方式

### 4. 生态板块
- 七大核心板块展示
- 生态体系概述
- 发展历程时间线
- 合作伙伴展示
- 生态愿景

## 技术栈

- React 18
- React Router 6
- TailwindCSS 3
- ECharts 5
- Vite 5

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

## 功能说明

### 登录系统
- 课程资源和认证体系需要登录才能访问
- 登录状态在导航栏显示
- 支持用户名密码登录

### 数据看板
- 使用ECharts实现中国地图可视化
- 三个合作等级通过颜色区分
- 点击省份卡片查看详细信息
- 学生按岗位类型分类统计

### 培训认证
- 分为在校学生和社会人员两个标签
- 社会人员可以提交岗位申请
- 培训模块显示时长、难度、学生数量
- 认证体系分为初级、中级、高级三个等级

### 需求互动
- 留言表单包含姓名、公司、联系方式、类型、内容
- 留言类型分为需求提交、意见反馈、咨询事宜
- 提交后显示成功提示
- 留言记录实时展示

### 生态板块
- 七大核心板块展示关键指标
- 发展历程以时间线形式呈现
- 合作伙伴以卡片形式展示
- 生态愿景展望未来发展方向
