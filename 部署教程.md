# 部署教程 - 如何把网站部署到Vercel让别人访问

## 第一步：准备GitHub账号

1. 访问 https://github.com/
2. 点击右上角 "Sign up" 注册账号
3. 填写邮箱、密码、用户名（建议用英文）
4. 验证邮箱
5. 登录你的GitHub账号

## 第二步：在GitHub上创建仓库

1. 登录后，点击右上角的 "+" 号
2. 选择 "New repository"
3. 填写仓库信息：
   - Repository name: 输入英文名称，比如 `neolink-education-platform`
   - Description: 描述（可选），比如 "无人车产业与教育融合平台"
   - 选择 "Public"（公开）
   - **不要勾选** "Initialize this repository with a README"
4. 点击 "Create repository" 按钮

## 第三步：上传代码到GitHub

在你的电脑上打开命令行（PowerShell或CMD），执行以下命令：

```bash
cd c:/Users/Administrator/CodeBuddy/20260320154309

# 复制刚才创建的仓库地址，类似这样：
git remote add origin https://github.com/你的用户名/仓库名.git

# 上传代码
git branch -M main
git push -u origin main
```

如果提示输入用户名和密码：
- 用户名：输入你的GitHub用户名
- 密码：需要使用GitHub的"Personal Access Token"（后面教你怎么获取）

### 如何获取GitHub Personal Access Token

1. 在GitHub点击右上角头像
2. 点击 "Settings"
3. 左侧菜单找到最下面的 "Developer settings"
4. 点击 "Personal access tokens" → "Tokens (classic)"
5. 点击 "Generate new token" → "Generate new token (classic)"
6. 填写：
   - Note: 随便写，比如 "vercel deploy"
   - Expiration: 选择 90 days 或 No expiration
   - 勾选 "repo" 权限
7. 点击 "Generate token"
8. 复制生成的token（只显示一次，记得保存）

然后用这个token代替密码输入。

## 第四步：注册并登录Vercel

1. 访问 https://vercel.com/
2. 点击 "Sign Up"
3. 选择 "Continue with GitHub"（用GitHub账号登录）
4. 授权Vercel访问你的GitHub
5. 填写用户名，点击 "Continue"

## 第五步：在Vercel部署项目

1. 登录后点击 "Add New" → "Project"
2. 你会看到你的GitHub仓库列表
3. 找到你刚才创建的仓库 `neolink-education-platform`，点击 "Import"
4. 配置项目（一般默认就行）：
   - Framework Preset: 自动识别为 "Vite"
   - Root Directory: `./`
   - Build Command: `npm run build`
   - Output Directory: `dist`
5. 点击 "Deploy" 按钮
6. 等待几分钟，Vercel会自动构建和部署

## 第六步：获取网站地址

1. 部署完成后，Vercel会给一个网址，类似：
   `https://neolink-education-platform.vercel.app`

2. 点击这个网址就能访问你的网站了

3. 把这个网址发给朋友，他们就能访问了

## 第七步：更新网站

如果修改了代码，更新步骤：

```bash
cd c:/Users/Administrator/CodeBuddy/20260320154309
git add .
git commit -m "更新内容描述"
git push
```

Vercel检测到代码更新会自动重新部署，几分钟就能看到新版本。

## 常见问题

### 1. 部署失败怎么办？
- 检查 package.json 里的 scripts 是否有 build 命令
- 检查 vite.config.js 配置是否正确
- 在Vercel查看具体的错误日志

### 2. 网站打不开？
- 等几分钟，Vercel需要时间部署
- 检查Vercel的部署状态，看是否有错误

### 3. 想要自定义域名？
- 在Vercel项目设置里可以添加自定义域名
- 需要购买域名（可以在阿里云、腾讯云等购买）
- 按照Vercel的指引配置DNS

### 4. 免费额度够用吗？

- 包括：无限带宽、100GB/月数据传输、自动HTTPS、全球CDN
